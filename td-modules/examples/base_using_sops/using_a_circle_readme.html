Carousel Illusions with a Cirlce SOP

It's sometimes prefereable to have endlessly cycling image carosles for installations or experiencs. Here the idea is that our screen is never empty, instead we always cycle back to the first image in the stack. This can be challengeing to approach with a simple line besed instancing techqique. Alternativly, we might think of this probelm as one that can be solved with some clever rendering tricks.

In this example we use a tube to describe the the positions of our iamges. A Tube SOP already has normals that face outward, allowing us to skip the process of changing the oritentaiotn of our normals. A delete SOP allow us to remove the extra row of poitns from geometry. We convert these to CHOP channels, and then use them to instance our photos. The real trick here is that we set our camera to be orthographic. We can then rotate the camera around the Y axis - the final illusion is that we have an endless image carosel that has required minimul effort to set-up and prep.
